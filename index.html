NLML-HPE Real-Time Head Pose Estimation Web Demo

This demo runs entirely in your browser using ONNX Runtime Web. It estimates yaw, pitch, and roll angles from 468 facial landmarks using our custom-trained neural network. All processing is done locally and no data is sent to any server.

Input

Output

Powered by ONNX Runtime Web for model inference and MediaPipe FaceMesh for facial landmark detection. The model was trained to predict head pose angles (yaw, pitch, roll) from 468 landmarks and converted to ONNX for efficient web deployment.

Please use a Chromium-based browser (e.g., Chrome or Edge) for best performance.
